# 🏎️ CarVault: Multi-City Car Search Engine

CarVault is a high-performance web application designed to help users find the best used car deals across multiple Indian cities. It aggregates data from various platforms into a single, clean, and mobile-responsive interface.

## ✨ Features
* **Multi-City Support:** Seamlessly switch between Mumbai, Ahmedabad, Pune, and more.
* **Smart Filters:** Filter by Budget (Lakhs), Fuel Type (Petrol/Diesel/CNG), and Transmission (Manual/Automatic).
* **Smooth UI:** Powered by Streamlit Forms to prevent page flickering during filtering.
* **Mobile Ready:** Fully responsive design for browsing on the go.
* **Direct Links:** One-click access to the original listing for full details.

## 🛠️ Tech Stack
* **Language:** Python 3.11
* **Frontend:** Streamlit
* **Data Processing:** Pandas
* **Scraping Engine:** Playwright (Stealth Mode)

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/your-username/car-vault.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
